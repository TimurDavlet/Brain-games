# Description

Brain-games - это 5 математических игр, которые вы сможете запустить у себя в терминале используя Node.js.
Список доступных игр:
  - brain-calc
  - brain-even
  - brain-gcd
  - brain-prime
  - brain-progression
 
# Setup
Brain-games requires [Node.js](https://nodejs.org/) v4+ to run.

Install:

```sh
$ make install
```

Play:
```sh
$ brain-calc
$ brain-even
$ brain-gcd
$ brain-prime
$ brain-progression
```
